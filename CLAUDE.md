# CLAUDE.md

Read and follow [AGENTS.md](./AGENTS.md). It is the canonical contributor and
agent instruction file for this repository.

Two rules worth repeating:

- `docs/` is the published website (<https://bobcli.dev>). Do not write
  anything there — notes, plans, scratch files, reports — unless it is meant to
  ship on the site.
- Run `bob learn --json` (or read <https://bobcli.dev/agents>) before driving
  the Bob CLI.
